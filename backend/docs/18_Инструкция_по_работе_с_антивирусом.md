# Инструкция по работе с корпоративным антивирусом (Kaspersky Endpoint)
**Версия:** 1.1 | **Дата обновления:** 05.05.2026 | **Ответственный:** ИБ-отдел

## 1. Основные функции
- Защита в реальном времени (файлы, сеть, веб-трафик).
- Контроль устройств (USB, Bluetooth, внешние диски).
- Шифрование съёмных носителей.

## 2. Повседневное использование
- Антивирус работает в фоновом режиме. Не отключайте защиту самостоятельно.
- При обнаружении угрозы появится уведомление. Следуйте инструкциям на экране.
- Обновления баз сигнатур приходят автоматически каждые 4 часа.

## 3. Ручная проверка
1. Откройте Kaspersky → «Проверка» → «Выборочная».
2. Выберите папку или файл.
3. Запустите. Результат появится в журнале событий.
4. При обнаружении угроз нажмите «Обезвредить» и уведомите IT.

## 4. Исключения и проблемы
- Добавление в исключения возможно только через заявку в Service Desk с обоснованием.
- Если программа «тормозит систему» → создайте тикет с логом `KAVDiag`.
- Запрещено устанавливать сторонние антивирусы поверх корпоративного.
