# Гайд по настройке корпоративного VPN для удалённой работы
**Версия:** 1.2 | **Дата обновления:** 05.05.2026 | **Ответственный:** Сетевые инженеры

## 1. Требования к ПО
- Windows 10/11: GlobalProtect или OpenConnect
- macOS: GlobalProtect (App Store) или Tunnelblick
- Linux: OpenConnect + NetworkManager

## 2. Пошаговая установка
1. Скачайте клиент по ссылке во внутренней инструкции или запросите пакет у IT.
2. Установите приложение, перезагрузите компьютер.
3. Запустите клиент и введите адрес шлюза: `vpn.company.ru`
4. Авторизуйтесь под своим корпоративным логином и паролем.
5. Пройдите 2FA через Authenticator.
6. Дождитесь статуса «Connected». Проверьте доступ к `confluence.company.ru`.

## 3. Типовые проблемы
| Симптом | Решение |
|---------|---------|
| Ошибка «Сервер недоступен» | Проверьте интернет, отключите сторонние VPN/AdBlock |
| Зацикливание на вводе пароля | Очистите кэш клиента, переустановите, проверьте время системы |
| Нет доступа к внутренним ресурсам | Убедитесь, что профиль сети установлен на «Частная», перезапустите клиент |

## 4. Безопасность
- VPN предназначен только для рабочих задач.
- Запрещён доступ с публичных сетей без дополнительного шифрования.
- Сессия автоматически завершается через 12 часов бездействия.
