# Инструкция по установке и обновлению корпоративного ПО
**Версия:** 1.0 | **Дата обновления:** 05.05.2026 | **Ответственный:** IT-отдел

## 1. Каталог разрешённого ПО
- Офисный пакет: Microsoft 365
- Антивирус: Kaspersky Endpoint Security
- Коммуникации: Outlook, Mattermost, Zoom
- Разработка/Дизайн: VS Code, Figma, Postman (по заявке)

## 2. Процесс установки
1. Откройте «Компания Центр ПО» (ярлык на рабочем столе или `https://software.company.ru`).
2. Авторизуйтесь под своим аккаунтом.
3. Выберите программу → нажмите «Установить».
4. Следуйте инструкциям мастера. Перезагрузка требуется только для системных обновлений.

## 3. Самостоятельная установка стороннего ПО
- Запрещена без письменного согласования IT-отдела.
- Для тестирования используйте виртуальную среду или sandbox.
- Нарушение политики влечёт отзыв прав локального администратора.

## 4. Обновления
- Критические обновления безопасности устанавливаются автоматически в нерабочее время.
- Плановые обновления уведомляются за 3 дня через `#it-announcements`.
