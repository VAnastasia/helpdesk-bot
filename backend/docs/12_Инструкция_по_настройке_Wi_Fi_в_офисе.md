# Инструкция по подключению к корпоративной сети Wi-Fi
**Версия:** 1.0 | **Дата обновления:** 05.05.2026 | **Ответственный:** Сетевые инженеры

## 1. Сети в офисе
| SSID | Назначение | Доступ |
|------|------------|--------|
| `CorpWiFi-Secure` | Основной рабочий | Только корпоративные устройства (802.1x) |
| `CorpWiFi-Guest` | Гости, партнёры | Временный доступ через портал регистрации |
| `IoT-Network` | Принтеры, датчики | Не для сотрудников |

## 2. Подключение Windows/macOS
1. Выберите `CorpWiFi-Secure`.
2. Введите логин `company\username` и пароль.
3. Примите сертификат доверия (выдан `CA-Company-Internal`).
4. Дождитесь статуса «Подключено, защищено».

## 3. Если подключение не удаётся
- Убедитесь, что устройство добавлено в AD.
- Отключите «Случайные MAC-адреса» в настройках Wi-Fi.
- Перезагрузите сетевой адаптер.
- При ошибке аутентификации создайте заявку с логом: `netsh wlan show wlanreport`.

## 4. Запреты
- Запрещено подключаться к `CorpWiFi-Guest` с корпоративных ноутбуков.
- Запрещено раздавать интернет с рабочих устройств.
