# Политика использования личных устройств (BYOD)
**Версия:** 1.0 | **Дата обновления:** 05.05.2026 | **Ответственный:** IT + ИБ

## 1. Область действия
Разрешает доступ к корпоративной почте и чату с личных смартфонов/планшетов при соблюдении условий.

## 2. Требования к устройству
- ОС: iOS 15+ / Android 11+ / Windows 10+
- Включён блокировщик экрана (PIN/биометрия)
- Установлен MDM-профиль компании
- Шифрование диска активно

## 3. Ограничения доступа
- Разрешено: Почта, чат, календарь, корпоративный диск.
- Запрещено: VPN, финансовые системы, CRM, внутренние порталы.
- При утере устройства IT удалит только корпоративные данные (контейнер MDM).

## 4. Отказ от политики
- Сотрудник вправе отозвать MDM-профиль в любое время.
- После отзыва доступ к рабочим сервисам с личного устройства блокируется.
- Компания не несёт ответственности за личные данные на устройстве.
