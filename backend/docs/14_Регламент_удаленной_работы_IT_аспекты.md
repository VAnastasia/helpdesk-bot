# Регламент удалённой работы: IT-требования
**Версия:** 1.1 | **Дата обновления:** 05.05.2026 | **Ответственный:** IT-директор

## 1. Требования к домашнему окружению
- Стабильный интернет ≥ 20 Мбит/с (проводное соединение предпочтительно).
- Отдельное рабочее место с блокировкой экрана от посторонних.
- Источник бесперебойного питания (рекомендуется для критичных ролей).

## 2. Программное обеспечение
- Обязательное ПО: VPN-клиент, антивирус, корпоративный чат, почта.
- Запрещено отключать MDM-агента или брандмауэр.
- Обновления ОС устанавливаются автоматически в окно 20:00–22:00.

## 3. Безопасность данных
- Не печатайте конфиденциальные документы дома без согласования.
- Используйте корпоративный аккаунт для всех рабочих сервисов.
- При работе в общественных местах обязательна активация VPN и приватного режима экрана.

## 4. Поддержка
- IT-отдел оказывает удалённую помощь через TeamViewer/AnyDesk (по заявке).
- Выезд к сотруднику на дом не предусмотрен. При поломке устройство отправляется курьером в офис.
