# Базовые правила информационной безопасности
**Версия:** 2.0 | **Дата обновления:** 05.05.2026 | **Ответственный:** CISO

## 1. Общие требования
- Все сотрудники обязаны проходить ежегодный тренинг по ИБ.
- Запрещён обход корпоративных защитных средств (антивирус, фаервол, DLP).
- Рабочие данные хранятся только на корпоративных ресурсах.

## 2. Пароли и доступы
- Пароль меняется каждые 90 дней.
- Запрещено использовать один пароль для работы и личных сервисов.
- При подозрении на компрометацию немедленно смените пароль и уведомите IT.

## 3. Работа с данными
- Файлы с пометкой «Коммерческая тайна» шифруются перед отправкой.
- Запрещено пересылать базы клиентов, финансовые отчёты, персональные данные на личную почту.
- Публичные Wi-Fi сети требуют обязательного включения VPN.

## 4. Реагирование на инциденты
1. Немедленно отключите устройство от сети.
2. Не перезагружайте и не чистите систему.
3. Сообщите в `security@company.ru` или по экстренной линии.
4. Следуйте инструкциям команды ИБ.
