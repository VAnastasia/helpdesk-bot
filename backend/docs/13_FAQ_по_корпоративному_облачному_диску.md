# FAQ по корпоративному облачному диску (SharePoint/OneDrive)
**Версия:** 1.2 | **Дата обновления:** 05.05.2026 | **Ответственный:** IT-отдел

## 1. Как получить доступ?
- Диск доступен автоматически через `https://onedrive.company.ru` или приложение OneDrive.
- Синхронизация с рабочим столом включена по умолчанию.

## 2. Как поделиться файлом?
- Нажмите «Поделиться» → выберите «Внутри компании» или «Конкретные люди».
- Избегайте ссылок «Все, у кого есть ссылка» для конфиденциальных данных.
- Срок действия ссылки: 30 дней (по умолчанию).

## 3. Восстановление удалённых файлов
- Корзина хранит файлы 93 дня.
- Версии файлов сохраняются до 500 штук (по 12 месяцев).
- Для восстановления обратитесь в IT, если файл удалён из корзины.

## 4. Лимиты
- Индивидуальный диск: 1 ТБ.
- Командные папки: 25 ТБ на сайт.
- Максимальный размер одного файла: 250 ГБ.
