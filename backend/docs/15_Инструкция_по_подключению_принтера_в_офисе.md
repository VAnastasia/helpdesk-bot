## 2. Подключение Windows
1. `Пуск → Параметры → Устройства → Принтеры и сканеры`.
2. «Добавить принтер» → «Добавить вручную по IP».
3. Введите IP-адрес, выберите протокол `TCP/IP`.
4. Установите драйвер из каталога ПО или скачайте с `software.company.ru`.
5. Распечатайте тестовую страницу.

## 3. Сканирование
- Используйте приложение «HP Scan» или `scan.company.ru`.
- Файлы сохраняются в папку `\\nas-scans\username`.
- Лимит: 50 страниц за раз, формат PDF/JPG.

## 4. Проблемы
| Симптом | Решение |
|---------|---------|
| «Принтер в автономном режиме» | Проверьте сеть, перезапустите службу печати |
| Ошибка драйвера | Удалите принтер, установите заново из каталога |
| Замятие бумаги | Откройте лоток, аккуратно удалите лист, закройте до щелчка |
