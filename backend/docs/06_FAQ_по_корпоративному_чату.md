# FAQ по корпоративному мессенджеру (Mattermost)
**Версия:** 1.3 | **Дата обновления:** 05.05.2026 | **Ответственный:** IT-отдел

## 1. Как получить доступ?
- Аккаунт создаётся автоматически при найме.
- Войдите через SSO: `chat.company.ru` → «Войти через Microsoft».
- Если доступ отсутствует, создайте заявку в категории «Учетные записи».

## 2. Как настроить уведомления?
- Настройки → Уведомления → выберите режим: «Все», «Упоминания», «Только DM».
- Можно отключить звук на определённое время («Не беспокоить»).

## 3. Как перенести историю и файлы?
- Экспорт разрешён только по согласованию с руководителем и IT.
- Файлы хранятся в чате до удаления. Для долгосрочного хранения используйте корпоративный диск.

## 4. Правила использования
- Не обсуждайте коммерческую тайну в открытых каналах.
- Не добавляйте внешних пользователей без разрешения безопасности.
- Каналы `#general`, `#announcements` предназначены только для официальных сообщений.
